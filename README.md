# 元素周期表前端

一个基于 `Vue 3 + Vite` 的科幻风元素周期表展示项目。当前版本采用单主卡片布局，上方包含像素终端风标题和元素粒子交互区，下方提供搜索、分类筛选、主题切换以及完整周期表展示。

## 功能特点

- 单卡片式主界面，视觉重心集中在元素周期表本体
- 顶部左侧为像素终端风标题，带命令行提示符与闪烁光标效果
- 顶部右侧为 118 个元素符号粒子动画区，支持点击扰动交互
- 支持 `6` 套主题风格切换
- 支持按元素名称、符号、原子序数搜索
- 支持按元素分类下拉筛选
- 悬浮元素卡片时，在表格附近显示详情浮层
- 包含镧系元素和锕系元素扩展区
- 适配桌面端和移动端布局

## 启动步骤

### 第 1 步：安装依赖

```bash
npm install
```

### 第 2 步：启动开发环境

```bash
npm run dev
```

### 第 3 步：在浏览器中访问

终端启动后，打开命令行里显示的本地地址，通常是：

```text
http://localhost:5173
```

## 交互说明

- 在顶部搜索框中输入元素名称、符号或原子序数，可快速过滤元素
- 使用下拉框切换元素分类
- 使用下拉框切换主题风格
- 点击顶部粒子区域，粒子会从点击位置向外扰动扩散
- 鼠标移动到周期表中的元素卡片上，会在附近显示元素详情卡片
